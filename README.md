# 📊 Telco Customer Churn: Full-Stack Data Analysis
**Predicting retention and calculating financial impact using Python, SQL, Excel, and Power BI.**

## 🚀 Project Overview
This project identifies at-risk customers for a telecommunications company and provides actionable business insights to reduce churn. It covers the entire data lifecycle from extraction to financial forecasting.

## 🛠️ Tech Stack
*   **Database:** SQL (PostgreSQL) for data extraction.
*   **Machine Learning:** Python (Scikit-Learn, Pandas) using Random Forest.
*   **Financial Modeling:** Excel for ROI and Campaign Cost analysis.
*   **Visualization:** Power BI (Dark Theme/UX focused).

## 📈 Key Results
*   **Model Performance:** 74% Accuracy with balanced Recall for churners.
*   **Top Drivers:** Tenure, Contract Type (Month-to-Month), and Total Charges.
*   **Business Impact:** Projected Net Profit of **₹3,98,500** by targeting high-risk customers.
